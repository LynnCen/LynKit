# useInterval

处理定时器的 Hook，自动清理。

## 使用场景

- 轮询请求
- 倒计时
- 动画效果

## 代码演示

### 基础用法

简单的计数器示例。

```tsx preview
import { useInterval } from '@lynkit/hooks';
import { useState } from 'react';

export default () => {
  const [count, setCount] = useState(0);
  const [delay, setDelay] = useState<number | null>(1000);
  
  useInterval(() => {
    setCount(count + 1);
  }, delay);
  
  return (
    <div style={{ display: 'flex', flexDirection: 'column', gap: '12px' }}>
      <div style={{ fontSize: '24px', fontWeight: 'bold' }}>
        计数: {count}
      </div>
      <div style={{ display: 'flex', gap: '8px' }}>
        <button onClick={() => setDelay(delay ? null : 1000)}>
          {delay ? '暂停' : '开始'}
        </button>
        <button onClick={() => setCount(0)}>重置</button>
      </div>
    </div>
  );
};
```

## API

```typescript pure
useInterval(fn: () => void, delay: number | null)
```

### 参数

| 参数 | 说明 | 类型 |
|------|------|------|
| fn | 定时执行的函数 | `() => void` |
| delay | 间隔时间（毫秒），为 null 时暂停 | `number \| null` |
