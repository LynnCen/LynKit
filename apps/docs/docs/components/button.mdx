# Button 按钮

按钮用于开始一个即时操作。

## 何时使用

- 标记一个（或封装一组）操作命令
- 响应用户点击行为
- 触发相应的业务逻辑

## 代码演示

### 基础用法

最简单的按钮使用方式。

```tsx preview
import { Button } from '@lynkit/ui';

export default () => (
  <div style={{ display: 'flex', gap: '8px' }}>
    <Button type="primary">主按钮</Button>
    <Button>默认按钮</Button>
    <Button type="link">链接按钮</Button>
    <Button type="text">文本按钮</Button>
  </div>
);
```

### 按钮尺寸

支持大、中、小三种尺寸。

```tsx preview
import { Button } from '@lynkit/ui';

export default () => (
  <div style={{ display: 'flex', gap: '8px', alignItems: 'center' }}>
    <Button size="large" type="primary">大按钮</Button>
    <Button type="primary">默认按钮</Button>
    <Button size="small" type="primary">小按钮</Button>
  </div>
);
```

### 加载状态

添加 `loading` 属性即可让按钮处于加载状态。

```tsx preview
import { Button } from '@lynkit/ui';
import { useState } from 'react';

export default () => {
  const [loading, setLoading] = useState(false);

  return (
    <div style={{ display: 'flex', gap: '8px' }}>
      <Button type="primary" loading>加载中</Button>
      <Button
        type="primary"
        loading={loading}
        onClick={() => {
          setLoading(true);
          setTimeout(() => setLoading(false), 2000);
        }}
      >
        点击加载
      </Button>
    </div>
  );
};
```

### 禁用状态

添加 `disabled` 属性即可让按钮处于禁用状态。

```tsx preview
import { Button } from '@lynkit/ui';

export default () => (
  <div style={{ display: 'flex', gap: '8px' }}>
    <Button type="primary" disabled>禁用主按钮</Button>
    <Button disabled>禁用默认按钮</Button>
    <Button type="link" disabled>禁用链接按钮</Button>
  </div>
);
```

### 危险按钮

用于危险操作，如删除、重置等。

```tsx preview
import { Button } from '@lynkit/ui';

export default () => (
  <div style={{ display: 'flex', gap: '8px' }}>
    <Button type="primary" danger>危险主按钮</Button>
    <Button danger>危险默认按钮</Button>
    <Button type="link" danger>危险链接按钮</Button>
  </div>
);
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| type | 按钮类型 | `'primary' \| 'default' \| 'link' \| 'text' \| 'icon'` | `'default'` |
| size | 按钮大小 | `'large' \| 'middle' \| 'small'` | `'middle'` |
| loading | 加载状态 | `boolean` | `false` |
| disabled | 禁用状态 | `boolean` | `false` |
| danger | 危险按钮 | `boolean` | `false` |
| ghost | 幽灵按钮 | `boolean` | `false` |
| icon | 按钮图标 | `ReactNode` | - |
| onClick | 点击回调 | `(e: MouseEvent) => void` | - |

## 设计指南

1. **主按钮**：一个页面中最多出现一次，用于最重要的操作
2. **默认按钮**：用于没有主次之分的一组操作
3. **危险按钮**：删除等危险操作，建议配合二次确认
