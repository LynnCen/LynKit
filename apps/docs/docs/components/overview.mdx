# 组件总览

LynKit UI 提供了一系列精心设计的 React 组件。

## 通用组件

| 组件 | 说明 |
|------|------|
| [Button](/components/button) | 按钮，用于触发操作 |

## 数据录入

| 组件 | 说明 |
|------|------|
| [Input](/components/input) | 输入框 |
| [Select](/components/select) | 下拉选择器 |

## 安装

```bash
pnpm add @lynkit/ui
```

## 使用

```tsx pure
import { Button, Input } from '@lynkit/ui';

function App() {
  return (
    <div>
      <Input placeholder="请输入" />
      <Button type="primary">提交</Button>
    </div>
  );
}
```
