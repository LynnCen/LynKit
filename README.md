# Lynkit UI

一个现代化的 React 组件库，基于 TypeScript 开发。

## 🌟 特性

- 📦 基于 TypeScript 开发，提供完整的类型定义
- 🎨 现代化的 UI 设计
- 🚀 支持按需加载
- 💪 使用 Monorepo 架构，便于维护和扩展
- 📱 支持响应式设计


## 📚 包结构

本项目包含以下几个核心包：

- `@lynkit/ui`: UI 组件库
- `@lynkit/hooks`: React Hooks 工具集
- `@lynkit/icons`: 图标库
- `@lynkit/api`: API 工具集

## 🔧 本地开发

### 环境准备

- Node.js >= 16
- pnpm >= 8

## 📦 安装

```bash

 // 使用 npm
npm install @lynkit/ui

 // 使用 yarn
yarn add @lynkit/ui

// 使用 pnpm
pnpm add @lynkit/ui
```

## 📦 使用

```tsx
import { Button } from '@lynkit/ui'
```

## 📦 开发

```bash
pnpm start
```     

## 📦 构建

```bash
pnpm build
```


## 📦 发布

```bash
pnpm publish
```
